# Repository untuk Workshop "Introduction to SQL Fundamental"

Hi, data enthusiast! Selamat datang di Repository untuk Workshop Introduction to SQL Fundamentals, yang akan diselenggarakan pada hari Rabu, 21 Januari 2026 
pukul 20.00 WIB oleh Data Enthusiast Community. 

Workshop ini bertujuan memperkenalkan basic SQL dalam manajemen Big Data. Hal ini dikarenakan hampir semua tools yang dipakai Data Engineer (spt Airflow, GCP, Power BI) melibatkan bahasa SQL. 
Pada workshop ini, akan dipelajari tentang basic database, basic SQL, dan praktek SQL query.

Tools yang perlu diinstall cukup dua saja, yaitu:
1. PostgreSQL
Link download : https://www.enterprisedb.com/downloads/postgres-postgresql-downloads

2. DBeaver (Community Edition)
Link download : https://dbeaver.io/download/
