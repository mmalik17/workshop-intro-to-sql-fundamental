# workshop-intro-to-sql-fundamental
Repository for Workshop "Introduction to SQL Fundamental" by Data Enthusiast Community

Hi, data enthusiast! Selamat datang di Repository untuk Workshop Introduction to SQL Fundamentals, yang akan diselenggarakan pada hari Rabu, 21 Januari 2026 pukul 20.00 WIB. 

Workshop ini bertujuan memperkenalkan basic SQL dalam manajemen Big Data. Hal ini dikarenakan hampir semua tools yang dipakai Data Engineer (spt Airflow, GCP, Power BI) melibatkan bahasa SQL. 
Pada workshop ini, akan dipelajari tentang basic database, basic SQL, dan praktek SQL query.
